Form

Form Explanation :-
A form page built with HTML and CSS is a structured layout that allows users to input and submit information.Forms in HTML are structures used to collect user input. They consist of elements like text fields, checkboxes, radio buttons, dropdown menus, and buttons. The form tag encloses these input elements and includes attributes like action (specifies where to send the form data)

Table Explanation :-
Tables in HTML are structures to display data in rows and columns. They use tags like table, tr for rows, th for headers, and td for cells. Tables organize data into a grid format, making it readable and structured.
